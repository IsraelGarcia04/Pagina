El Internet es una red que se compone de ordenadores en todo el mundo. Ofrece a quien lo use infinitas posibilidades, entre ellas acceder a correos, juegos, blogs, entre otros. Para poder ejecutar esto hace uso de un método de funcionamiento llamado TCP/IP, que tendrá a cargo poder recibir y mandar información.

Este método debe tomar en cuenta una serie de normas, las cuales debe cumplir para que sea posible el intercambio de información entre los ordenadores.

Por medio de la IP se va a identificar al ordenador para que este pueda comunicarse con dos operadores, de esta forma estos se conectan entre ellos con sus direcciones IP respectivas. Para los seres humanos es complicado recordar los IPs, por lo que resulta más factible memorizar nombres de dominios. La traducción se hace internamente, por medio de los servidores DNS.

  • DNS

    El DNS significa en castellano sistema de nombre de dominio. Su función principal será traducir o resolver aquellos nombres inteligibles para los usuarios en identificadores binarios relacionados con los aparatos conectados a la red.

    Sin el DNS, quien use Internet tendría que entrar a una página web usando la dirección IP, como http://214.87.34.234 en lugar de http://undominio.com.

  • Dominio

    El dominio de Internet es el nombre que se le da a una dirección de computadora conectada a la red. Es más fácil de recordar que una IP. Gracias a él y a los DNS, se pueden traducir las direcciones IP.

    El dominio permite trasladar un servicio de red entre ubicaciones sin cambiar la IP. Las IP pueden ser dinámicas (cambian con frecuencia) o estáticas (fijas), necesarias para servidores web, DNS, FTP, etc.

  • HTTP

    Significa HyperText Transfer Protocol. Es el protocolo que se utiliza para las transacciones en la web. El hipertexto es el contenido y el protocolo de transferencia es el sistema que envía peticiones y recibe respuestas.

    También permite enviar datos en ambos sentidos, por ejemplo, mediante formularios.

  • WWW

    Las tres W hacen referencia a World Wide Web. Es el sistema de navegación encargado de extraer la información de las páginas web.

    "Una web" se refiere a un sitio específico; "la web" a toda la red interconectada.

  • .com

    Es un dominio genérico de Internet. Surgió en 1985 y lo administra VeriSign. Es uno de los más usados, especialmente por negocios en línea.

  • Página web

    Una página web contiene texto, imágenes, aplicaciones y otros contenidos accesibles desde un navegador web.

  • Correo electrónico

    También llamado email, es un servicio para enviar y recibir mensajes de forma segura y rápida por medio de un canal electrónico.

    Permite la comunicación entre dos o más personas mediante dispositivos o computadoras.

  • Navegador web

    Es un programa que permite a los usuarios acceder a la información de páginas web. Interpreta el código HTML y lo muestra en pantalla de forma entendible.

Conocer cuáles son los componentes de red te resultará de utilidad para identificar posibles problemas de conexión y descubrir soluciones para optimizar el funcionamiento de las redes, asegurando que cumplan su propósito de forma adecuada. A continuación, se describen los principales componentes de redes.

El servidor

Es una computadora o aplicación encargada de proporcionar servicios a otros equipos denominados clientes. Procesa solicitudes y entrega datos por medio de una red, ya sea local o mediante Internet.

  • Ejemplos: servidores de correo electrónico y servidores web.
  • Funciona bajo la arquitectura cliente-servidor.

La estación de trabajo

Se refiere al equipo informático utilizado para tareas profesionales, técnicas o científicas. En oficinas, suelen estar conectadas entre sí y a un servidor, así como a periféricos y a Internet.

  • Facilitan el flujo de información.
  • Hoy en día, no existen grandes diferencias entre estaciones de trabajo y equipos domésticos.

La tarjeta de conexión a la red

Componente generalmente integrado en los equipos, permite la conexión a redes mediante controladores compatibles con protocolos del sistema operativo.

El repetidor

Retransmite señales de red débiles, amplificándolas para mejorar su alcance y potencia. Su función depende del tipo de red (cableada o inalámbrica).

  • Útil en entornos con muchos dispositivos conectados.

Los bridges o puentes de red

Dispositivos que interconectan segmentos de red, creando una subred sin necesidad de router. Permiten la comunicación entre diferentes equipos mediante conexiones segmentadas.

Los hubs

Dispositivos que conectan múltiples equipos mediante cables, creando un único segmento de red. Funcionan como concentradores, compartiendo información simultáneamente entre todos los puertos conectados.

  • Conectan distintos dispositivos, independientemente de su tipo.

El switch

Dispositivo digital que interconecta segmentos de red, similar a los puentes. Permite la creación de redes de área local (LAN) bajo el estándar Ethernet.

El router

También denominado “enrutador”, es responsable de dirigir la información por la mejor ruta posible en una red. Su uso más común es compartir una única conexión a Internet entre varios dispositivos.

  • Recibe y distribuye la conexión de red.
  • Optimiza el tiempo y calidad de entrega de datos.

Access Point

Elemento esencial en redes inalámbricas. Recibe la señal del router y extiende su cobertura generando una nueva señal independiente.

  • Puede configurarse como repetidor o puente.
  • Comúnmente utilizado en redes empresariales.
